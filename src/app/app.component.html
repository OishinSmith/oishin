Software Developer

January 2021 - Present

Jigsaw Inusrance PLC

Technology used: PHP | Laravel | Angular | AWS | SQL | Git | Github | Docker | HTML | CSS | JQuery | Javascript

I'm currently working Jigsaw Inurance, who are an insruance company and have been working there for more than... two years. Our software development team is made up of five developers. we're an agile team who work on two week sprints. Planning normally starts on the monday where we discuss the sprint objective, tasks, stories, bugs and assign them to eachother. At the end of our sprint we have our sprint retrospective where we talk about the positives, negatives and possible improvements of the past sprint. Furthermore, we're a team who strongly follow the software development lifecycle which involves requirements gathering, Design, Development, Testing, Deployment and Maintenance of software.

My day to day responsabilities mostly/may involve back-end and front-end development, DevOps, quality assurance, project documentation and project management. Every sprint I will develop software depending on the requirements. I will then thoroughly test the software locally and on our development environments making sure the requirements are met for the new software before presenting it to my development team and the business.

Once the new software has been presented, I will start the release process. This involves creating documentation/release-notes for the new software, creating a release branch and a sprint task in Jira to release the new software and also tagging the tickets in Jira with the appropriate release version. When we're ready to release the new software I will create an email and attach the release notes to it and communicate to the business that new software is going to be released at a specific time. When it is time, I will release the software into production, do some integration testing to see if the new features/bug-fixes are available and monitor the websites/software to check for errors.
